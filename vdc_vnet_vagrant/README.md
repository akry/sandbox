vdc_vnet_vagrant
=====================

System Requirements
-------------------

+ [Vagrant](http://www.vagrantup.com/downloads.html)
+ [VirtualBox](https://www.virtualbox.org/wiki/Downloads)

Getting Started
---------------

copy RSA keyfiles to `guestroot/.ssh/` then

```
$ vagrant up
```
